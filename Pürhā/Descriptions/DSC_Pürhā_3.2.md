# Concept 3.2 — Minimalist Graphic / Pastel Tones + Editorial Typography

* **The Vibe:** Premium Swiss design. It feels like an official catalog for a contemporary architecture exhibition or a high-end art book (Phaidon/Taschen style).
* **Colors & Style:** Shares the same sophisticated pastel palette and clean geometric silhouettes as 3.1, but elevates the layout to a rigid design system.
* **The Typography:** The name "Pürhā" stands out as an independent architectural element. It uses an ultra-precise, high-contrast Swiss sans-serif (like Neue Haas Grotesk) or a razor-thin serif, aligned with mathematical precision along the margins.
* **Why it's cool:** The rigid, authoritative typography applied over soft pastel shapes signals absolute creative seriousness. It stops looking like a "cute illustration" and becomes a high-end collectible design object.
